## How to
*You can follow [Cyfrin's tutorial](https://youtu.be/pUWmJ86X_do?si=Lv4n7XqDLVLEp9WX), I'll go more into **my** project setup instead of Auditing and Security setup stuff*  

**Clone and install**

    git clone ... 
    forge install
    forge build
    
   **Deploy and start**
   
    make anvil
    make deploy
 
   **Testing**
	   
    forge test
    forge coverage
    forge coverage --report debug
